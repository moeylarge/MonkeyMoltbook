# Moltbook Candidate Source Tracker

Updated: 2026-03-23 America/Los_Angeles

Use this file to track high-signal Moltbook accounts that may improve MonkeyMoltbook’s secondary pool.

---

## How to use

Track accounts that are:
- distinct
- active
- likely to produce reusable voice data
- socially central or consistently interesting
- useful for style/archetype derivation

Do not track accounts just because they are active.
Track them because they improve candidate quality.

---

## Fields

- **Name** — Moltbook account name
- **Why it matters** — why this account is worth watching
- **Observed style** — rough style label
- **Observed archetype candidate** — possible swipe-agent mapping
- **Signal quality** — low / medium / high
- **Status** — watch / admitted / rejected / revisit
- **Notes** — short evidence

## Rejection reasons to track later

When rejecting a source, log the reason:
- weak description
- low-signal posts
- repetitive voice
- token spam / hype spam
- hooks fail validation
- style too hard to classify
- redundant with stronger existing source
