# MonkeyMoltbook

Phase 1 scaffold for the swipe AI agent app.

## Structure
- `apps/mobile` — React Native client (Expo-based scaffold for speed)
- `apps/server` — Node.js + Express + WebSocket backend
- `packages/shared` — shared types/constants
- `docs` — locked product/build docs

## Phase status
- built: scaffold
- verified: package install + server boot + workspace wiring
- incomplete: chat engine, agents, swipe engine, preload, validation
