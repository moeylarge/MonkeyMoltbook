# NOT A COUPLE — PILOT BUILD SHEET

## Purpose

This file converts the current 7-shot visual board into a producible pilot sequence.

It maps:
- shot order
- story beat
- emotional function
- likely dialogue function
- editorial purpose

This is the bridge from asset generation into actual episode assembly.

---

## Pilot premise

At Central Peak, the group notices a **Couples Night** promo.
That immediately causes everyone to get weird about who is or is not a couple.

The tone should feel:
- warm
- funny
- socially charged
- character-driven
- comfort-sitcom, not sketch-comedy chaos

---

## Core sequence

1. establish the world
2. reveal the trigger
3. show group reaction
4. isolate Mon + Chance tension
5. isolate Russ + Rachelle awkwardness
6. isolate Jojo + Fufu comedy-chaos
7. land the ensemble payoff

## SHOT 3 — Group notices the sign

### Asset
`03-group-notices-sign`

### Story beat
Everyone reacts differently.

### Function
- introduces all six response lanes
- establishes that the sign has social consequences
- kicks off the ensemble comedy engine

### Likely duration
- 1.5 to 2.5 seconds

### Dialogue function

## SHOT 4 — Mon + Chance beat

### Asset
`04-mon-chance`

### Story beat
The strongest banter/chemistry lane surfaces.

### Function
- introduces the “obvious-not-obvious” romantic tension
- creates tension without explicit confirmation
- gives the pilot edge and charge

### Likely duration
- 1.5 to 2.5 seconds

### Dialogue function

# DIALOGUE / SCRIPT MATCHING RULE

When matching script to shots:
- assign only one main emotional beat per shot
- do not overload a frame with too many ideas
- let the pair shots carry character-specific lines
- let Shot 7 carry the button or escalation

---

# IF SOMETHING IS MISSING

Only add new shots if one of these problems appears:
- transition from Shot 3 to Shot 4 feels abrupt
- the sign is not legible enough
- Shot 7 does not feel like a real payoff
- pair shots need tighter line-specific inserts

If so, add only one of these:
- bridge shot: group clustering around couch/table
- tighter sign insert
- final punch-in reaction shot

Do not add filler shots.
