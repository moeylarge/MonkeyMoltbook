# NOT A COUPLE — DIALOGUE / LINE PLACEMENT

## Purpose

This file maps the current 7-shot pilot board to a first-pass dialogue rhythm.

The goal is not final script perfection.
The goal is to determine:
- what each shot is saying
- who owns each beat
- how the sequence breathes
- where the joke/tension actually lands

---

## Tone rule

Dialogue should feel:
- warm
- character-driven
- socially charged
- lightly heightened
- comfort-sitcom funny

Avoid:
- over-explaining
- too many words per shot
- trying to make every line a joke
- generic Marvel banter rhythm

---

## Sequence overview

1. Central Peak establishing
2. Couples Night sign
3. group notices sign
4. Mon + Chance beat
5. Russ + Rachelle beat
6. Jojo + Fufu beat
7. ensemble payoff

# FIRST-PASS LINE PLACEMENT

## SHOT 1 — Central Peak establishing

### Visual function
Return to the world.

### Dialogue approach
Optional ambient opening only.
This shot can be silent except for cafe ambience, or carry the start of an off-screen read.

### Best use
- soft ambience
- music sting
- or off-screen line starting just before Shot 2

### Recommended choice
Keep this mostly silent.
Let the world breathe for a beat.

---

## SHOT 2 — Couples Night sign

### Visual function
Show the episode trigger.

### Line owner
**Jojo** or **Fufu**

### Recommended line
**Jojo:** “Couples Night?”

### Why
- simple
- readable
- lets the premise land cleanly
- Jojo being the one to read it gives the moment softness

### Alternate line
**Fufu:** “Oh, this is going to ruin everybody.”

Use only if you want faster escalation.

---

## SHOT 3 — Group notices sign

### Visual function
Show all six reactions starting to split apart.

### Line owner
**Fufu**

### Recommended line
**Fufu:** “No, no — let’s not pretend this isn’t for us.”

### Why
- immediately escalates
- creates social discomfort
- feels like Fufu lighting the fuse

### Background reaction possibilities
- Mon: visible tension
- Chance: amused look
- Russ: panic thought starting

### Alternate version
**Fufu:** “This is amazing. I can feel three emergencies already.”

Use if you want her more chaotic/funny.

---

## SHOT 4 — Mon + Chance

### Visual function
Main chemistry engine enters.

### Line owner
**Chance**

### Recommended line
**Chance:** “You look tense for someone who’s definitely not part of a couple.”

### Mon response option
**Mon:** “I’m tense because your face is doing that.”

### Why this works
- Chance pokes the bruise
- Mon denies by redirecting
- chemistry becomes visible without explicit confession

### Editorial recommendation
If only one line fits here, use Chance’s line.
If two fit, add Mon’s response.

---

## SHOT 5 — Russ + Rachelle

### Visual function
Awkward romantic engine.

### Line owner
**Russ**

### Recommended line
**Russ:** “Just so we’re all being normal, I’d like to say this sign is weirdly aggressive.”

### Rachelle response option
**Rachelle:** “You say that like it personally invited you.”

### Why this works
- Russ tries to sound rational
- instantly reveals himself instead
- Rachelle clocks it with amused precision

### Editorial recommendation
If only one line fits here, use Russ.
If two fit, Rachelle’s reply makes the beat land better.

---

## SHOT 6 — Jojo + Fufu

### Visual function
Comedy-chaos pair lifts the sequence.

### Line owner
**Fufu**

### Recommended line
**Fufu:** “I can name two couples and one cover-up right now.”

### Jojo response option
**Jojo:** “Why is that worse than if you named three couples?”

### Why this works
- Fufu escalates recklessly
- Jojo grounds the scene with confused sincerity
- gives the pair their proper rhythm

### Editorial recommendation
If only one line fits here, use Fufu.
If two fit, Jojo’s question gives the beat extra life.

---

## SHOT 7 — Ensemble payoff / button

### Visual function
Land the whole premise.

### Best line owner
**Chance** or **Fufu**

### Recommended final button
**Chance:** “Relax. If anyone asks, I’ll claim Mon for the discount.”

### Why this is the best current button
- directly pays off the episode premise
- sharpens Mon + Chance
- turns the whole room unstable at once
- feels like a clean ending line

### Alternate chaos button
**Fufu:** “Great. We’re assigning couples now. This is my Olympics.”

Use only if you want a more chaotic final button instead of a romantic-comedy one.

# PERFORMANCE NOTES

## Mon
- sharp
- controlled
- reaction before words matters
- never too broad

## Chance
- amused, dry, unhurried
- should sound like he enjoys destabilizing the room

## Russ
- earnest and unintentionally revealing
- should sound like he’s trying to be normal and failing

## Rachelle
- clean, socially precise
- not cruel, just observant

## Jojo
- warm confusion
- sincere reactions are funnier than cleverness

## Fufu
- biggest escalation energy
- should sound delighted by the unfolding chaos

---

# TIMING RULE

Do not cram every line tightly.
Leave room for:
- reaction faces
- awkward silence
- eye contact
- social discomfort

The sequence should feel like people reacting to each other, not racing through punchlines.
