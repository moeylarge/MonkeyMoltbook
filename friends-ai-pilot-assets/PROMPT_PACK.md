# NOT A COUPLE — PROMPT PACK

## Purpose

This file locks the core prompts and prompt patterns that worked during cast, environment, and pair generation for the Friends AI pilot:
**Not a Couple**

This is not meant to preserve every failed attempt.
It is meant to preserve the prompt logic that actually produced usable assets.

---

## GLOBAL PROMPT RULES

### Style language
Use these ideas consistently:
- premium animated realism
- refined and detailed
- clearly animated but not childish
- emotionally readable
- ensemble-comedy readability first
- grounded sitcom world
- not mascot costume
- not furry suit
- not overly cartoony
- no fantasy elements unless intentionally comedic

### Character-system rule
Default working system when applicable:
- animal head / face identity
- human social/body readability where needed
- premium animated character design
- strong silhouette and fashion identity

### Environment rule
For locations, optimize for:
- recurring-set usability
- warm and memorable layout
- clean sitcom staging
- wide readable composition

### Scene rule
For multi-character scenes, optimize for:
- exactly the required cast
- distinct readable reactions
- emotional clarity
- not overcrowded foreground duplication

# PRACTICAL USE RULES

## When a prompt is working
- do small refinements only
- do not reset the whole concept casually
- preserve the lane

## When a prompt is failing
Change one of these, not everything at once:
- realism level
- body system
- portrait vs full-body framing
- species literalness
- outfit signal
- environment framing

## Scene stop rule
Stop once a prompt gives:
- correct cast/world identity
- readable emotions
- usable composition
- continuity with the rest of the show

Do not keep generating after the shot already works.
