# Mon + Chance Keeper Rubric v2

A candidate only passes if it clears all three gates below.

## Gate 1 — Character truth
Must pass all:
- Mon reads as Mon, not generic shy girl, streetwear teen, or mascot squirrel
- Chance reads as Chance, not bro-coded fox, smirking model, or soft himbo
- their expression relationship reads as tense, specific, and story-bearing

## Gate 2 — Scene continuity
Must pass all:
- still clearly feels like Central Peak
- keeps a medium two-shot intimacy
- preserves the pair geometry and conversational power balance
- does not replace the cafe beat with generic coffee-shop lifestyle imagery

## Gate 3 — Finish quality
Must pass all:
- premium anthropomorphic realism
- believable hands / paws / limbs
- no major anatomy drift
- no obvious costume drift away from approved silhouette
- no poster/promo over-stylization

## Fast reject triggers
Reject immediately if any of these appear:
- hats / hoodies / flannel makeover drift
- extra characters that matter compositionally
- wrong setting logic
- over-flirty romcom posing
- dead eyes or vague expressions
- mascot softness
- obvious hand failure

## Outcome labels
- **KEEPER** = clears all gates; can be upscaled or motion-tested
- **NEAR MISS** = strong enough to inform one surgical follow-up pass
- **REJECT** = do not animate; keep only as failure memory
