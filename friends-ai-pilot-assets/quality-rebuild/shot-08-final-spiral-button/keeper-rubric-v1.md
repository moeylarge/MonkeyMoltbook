# Shot 08 Final Spiral / Button — Keeper Rubric v1

A candidate only passes if it clears all three gates below.

## Gate 1 — Ensemble character truth
Must pass all:
- exactly six main characters, all readable
- Mon is clearly the most affected
- Chance reads calm, sly, and socially comfortable
- the rest of the ensemble each read as distinct reactions, not generic filler

## Gate 2 — Scene continuity
Must pass all:
- still clearly feels like Central Peak
- preserves the core couch / chair / table geography
- no background extras that materially change the room logic
- no readable signage or homage branding drift

## Gate 3 — Finish quality
Must pass all:
- premium anthropomorphic realism
- stable faces across the ensemble
- believable hands / paws / limbs
- no major anatomy drift
- final-frame quality high enough to justify motion testing

## Fast reject triggers
Reject immediately if any of these appear:
- extra background patrons
- fewer or more than six main characters
- duplicated or merged characters
- readable signage / logo drift
- Mon loses affected priority
- Chance loses sly-button energy
- warped hands or dead faces
- generic poster / promo feel instead of scene-specific sitcom frame

## Outcome labels
- **PASS / KEEPER** = clears all gates; eligible for one controlled motion pass
- **REJECT** = do not animate; preserve only as audit memory
