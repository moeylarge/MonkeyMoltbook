## Purpose

This document replaces the current "ship the rough cut" mindset.

The goal is to rebuild `Not a Couple` to the quality class of the TikTok reference John provided:
- watchable as a real AI-native episode
- visually intentional
- performance-led
- socially legible
- polished enough that we can confidently say it belongs in the same league before showing it again

This is a producer document.
It is not a rough assembly note.
It is not a patch list for the current broken export.

## Quality gate

Do **not** present another updated pilot as if it is done unless all of the following are true:

1. no obvious sync mismatch between voices and visual beats
2. no freeze-frame extension reads as a cheap patch
3. no shot outstays its energy
4. every voice feels plausibly cast for the role
5. the edit feels directed rather than mechanically assembled
6. the output is genuinely in the same quality class as the reference benchmark

If those are not true, say so plainly.

# Producer diagnosis — what the final-quality version must solve

## 1. The scene must feel authored, not assembled
That means:
- reaction timing matters as much as dialogue
- shot changes must follow tension/comedy
- pauses must feel motivated, not accidental

## 2. Every shot must be able to survive its runtime
A shot that only looks good for 3–5 seconds cannot be used for 8–10 seconds unless it is regenerated for that purpose.

## 3. The middle of the room must feel alive
The current scene concept is strongest when:
- Fufu destabilizes the room
- Mon/Chance become visible
- Russ/Rachelle reveal themselves socially
- the group reforms and spirals

The final cut must preserve that room dynamic without dead air.

## 4. The visual standard must rise to the reference class
Target quality means:
- cleaner final imagery
- more believable shot motion
- stronger shot-to-shot coherence
- no "patched AI slideshow" feeling

# Final shot package

This is the publish-oriented rebuild plan.

## Runtime target
**55–75 seconds**

Short enough to stay punchy.
Long enough to feel like a real scene.

## Final cut philosophy
- 1 establishing world shot
- 1 trigger insert
- 3 character-pair / reaction engines
- 1 regroup beat
- 1 final spiral/button beat
- optional 1 reaction insert if needed

**Target final shot count: 8 shots**

Not 15 tiny fragments.
Not 1 stretched montage.
A confident short scene.
