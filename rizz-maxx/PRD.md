# PRD.md

## Product
RIZZ MAXX

## Category
AI-powered dating profile optimizer mobile app.

## Core objective
Help users improve dating profile performance by ranking photos, identifying weak images, and delivering actionable recommendations that create a strong urge to improve and retest.

## MVP promise
Within minutes, a user can upload 4-10 photos, receive a ranked profile-strength report, understand their best lead photo and weakest photos, and see clear next actions.

## Product thesis
Users want validation, challenge, clarity, and improvement momentum more than pseudo-scientific "accuracy." The product should feel emotionally meaningful, visually premium, and practically useful.

## Target user
A user who wants more matches and a stronger dating profile without guessing which photos help or hurt.

## MVP features
1. Onboarding
2. Multi-photo upload flow
3. Photo ranking engine
4. Results screen with score, best/worst photo, strengths, weaknesses, and action plan
5. Improvement plan
6. Premium gate for deeper breakdown
7. Saved sessions
8. Basic settings/profile

## Explicit non-MVP
- dating marketplace
- swiping
- chat/messaging
- social graph
- public profiles
- communities
- AI texting coach
- full bio/prompt generator as core product
- voice mode
- complex referral/game economies

## Core free output
- overall profile score
- recommended lead photo
- strongest photos
- weakest photos
- top strengths
- top weaknesses
- key improvement actions

## Premium output
- detailed per-photo breakdown
- strongest 6-photo ordering
- deeper vibe/archetype report
- replacement guidance
- profile strategy suggestions

## UX standard
- premium
- dark-mode friendly
- emotionally engaging
- screenshot-worthy results
- App Store ready visual quality

## Credibility rule
Frame output around profile effectiveness and first-impression quality, not false claims of objective attractiveness truth or guaranteed match gains.

## MVP success standard
The app succeeds if the user quickly understands the value, the upload flow feels smooth, the result feels emotionally meaningful, and the premium gate feels desirable.
