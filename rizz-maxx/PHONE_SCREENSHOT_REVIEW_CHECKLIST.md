# RizzMaxx — Existing iPhone Screenshot Review Checklist

## Purpose
Review the 5 existing iPhone screenshot captures and decide, slot by slot, whether each one is:
- keep as-is
- keep with light recrop/polish
- replace with a fresh capture

This is a screenshot-quality review step only. It does not imply immediate App Store submission.

---

## Decision scale
For each screenshot, mark one:
- [ ] Keep as-is
- [ ] Keep with light polish
- [ ] Recapture

And score each on:
- clarity
- hierarchy
- premium feel
- App Store readability
- truthfulness to product value

Use a 1–5 scale mentally if needed. Anything below a strong 4 should be a recapture candidate.
