# FRIENDS AI — WARDROBE PLAN

## Purpose

This file locks the wardrobe strategy for the six-character ensemble.

The rule is simple:
- keep each current approved outfit as the **base fashion identity**
- do **not** redesign the cast broadly
- add only a small number of wardrobe variants per character
- outfits can change by scene, but each character’s fashion language must remain stable

---

## Global wardrobe rules

- each character keeps a distinct fashion identity
- current approved looks should be treated as the baseline
- add only **3 scene variants** beyond the base look:
  1. cafe look
  2. apartment look
  3. going-out look
- do not overbuild wardrobes before scene generation begins
- wardrobe should support sitcom readability first, novelty second
- all outfits should still feel like they belong in the same show universe
