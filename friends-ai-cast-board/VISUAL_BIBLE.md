# FRIENDS AI — VISUAL BIBLE

## Purpose

This file locks the first-pass visual identity of the main cast for Friends AI.

The goal is consistency:
- same show universe
- same rendering language
- same character logic
- distinct silhouettes and personalities
- outfits can change, but visual identity should remain stable

---

## GLOBAL STYLE RULES

- premium animated realism
- emotionally readable faces
- grounded sitcom-first readability
- animal identity must be clear, but personality matters more than zoological purity
- no mascot costume energy
- no furry-suit look
- no Disney-prince / princess defaulting
- no superhero styling
- no fantasy-princess drift unless explicitly intended for a joke/bit
- fashion identity is locked; outfits can change by scene
- characters should feel like they belong to the same ensemble show

### Character system

Default working system:
- animal head / face identity
- human social/body readability where useful
- premium animated character design
- realistic enough to feel premium
- animated enough to stay charming and controlled
