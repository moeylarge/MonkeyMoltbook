# MCA Funnel Wireframes

## MCA Funnel Wireframe 01 — Main landing page to short form
Landing Page
-> Above-fold CTA
-> Short Qualification Form
-> Thank-you / scoring pending
-> Lead in dashboard

Purpose:
Fastest path for high-intent inbound MCA leads.

## MCA Funnel Wireframe 02 — Niche page to qualifier
Niche Page
-> niche-specific CTA
-> Short Form
-> Optional second-step qualifiers
-> Lead scored
-> Lead detail review

Purpose:
Capture better-fit vertical leads.

## MCA Funnel Wireframe 03 — City page to qualification check
City Page
-> local CTA
-> Qualification Check Form
-> Score assigned
-> Dashboard queue

Purpose:
Capture geo intent and local relevance.
