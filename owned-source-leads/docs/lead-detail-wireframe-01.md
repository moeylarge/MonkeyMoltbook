# Lead Detail Wireframe 01

## Header
- Lead ID
- Vertical
- Lead Type
- Score
- Temperature
- Review Status
- Export Readiness

## Source block
- Exact source
- Page URL if inbound
- Source URL if prospecting
- Landing page slug
- Source bucket
- Submission or collection timestamp

## Collected fields block
- all captured fields shown in labeled rows

## Scoring block
- score
- temperature
- why it was scored that way
- why it was marked hot
- positive signals
- negative signals

## Consent block
- show for inbound debt and when applicable
- consent checkbox value
- consent text version
- consent timestamp

## Activity block
- attribution events
- review actions
- export history
