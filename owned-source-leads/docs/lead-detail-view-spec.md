# Lead Detail View Specification

## Required display fields
Each lead detail view must show:
- lead type
- vertical
- exact source
- page URL if inbound
- source URL if prospecting
- all collected fields
- score
- temperature
- why it was scored that way
- why it was marked hot
- consent data if inbound
- review status
- export readiness

## Required supporting sections
- source proof
- scoring summary
- hot reason codes
- duplicate status
- attribution events
- review history
- export history

## Required operator actions
- approve
- reject
- mark duplicate
- request enrichment
- mark export-ready
