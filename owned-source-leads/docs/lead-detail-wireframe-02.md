# Lead Detail Wireframe 02

## Two-column layout

### Left column
- lead summary card
- contact/business details
- qualification inputs
- consent details

### Right column
- source proof
- scoring explanation
- hot reason codes
- duplicate status
- review notes
- export readiness checklist

## Required shown fields
- lead type
- vertical
- exact source
- page URL if inbound
- source URL if prospecting
- all collected fields
- score
- temperature
- why scored that way
- why marked hot
- consent data if inbound
- review status
- export readiness
