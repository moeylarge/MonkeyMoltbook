# Dashboard Specification

## Required dashboard metrics
- new leads today
- hot leads today
- hot leads by vertical
- inbound hot leads
- prospecting hot leads
- leads by landing page
- leads by source
- duplicates removed
- junk leads
- last inbound submission time
- last collection time

## Required filters
- MCA / Debt
- inbound / prospecting
- hot / warm / cold / junk
- landing page
- source
- review status

## Required actions
- open lead detail
- filter hot queue
- review duplicates
- inspect missing consent
- inspect missing attribution
- inspect export-ready batches
